# Java Learning Roadmap
Моє навчання Java до рівня Junior Backend Developer (Spring).
План навчання включає:
- Java Core
- ООП та SOLID
- Алгоритми та структури даних
- Multithreading
- JVM (глибоке розуміння)
- Design Patterns
- Git та GitHub
- SQL
- Spring Boot
- Проєкти та підготовка до співбесід
